# Portfolio

My personal portfolio website. All due credits to [Mayank Agarwal](https://github.com/mayankagarwal09/dev-portfolio?tab=readme-ov-file), the original author of the code.

Check it out at [jsurrea.github.io](https://jsurrea.github.io)

## Setup 🔧

```bash
# Install dependencies
$ npm install --legacy-peer-deps

# Force audit fix
$ npm audit fix --force

# Start the development server
$ npm start
```

Once the server has started, go to this url `http://localhost:3000/` to see the portfolio locally.
